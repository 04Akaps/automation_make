# Project 설명

Make.com을 활용한 자동화 서비스를 도입하여, 데이터를 수집하고, 이를 제공하며 추가적으로 Stripe를 활용한 유료 결제 서비스
- 아직 한참 작업중인 Repository

# 기술 스펙
```
Stripe : PG 결제를 지원

TypeScript : 프로그래밍 언어

MySQL : 데이터 저장공간

RSS : 배치성 데이터 수집 및 실시간 데이터 수집

Make.com : 특정 업무 자동화를 위해 사용
```


# 프로젝트 관리
```
헥사고날 아키텔처를 기반으로 작성 진행 중
```
